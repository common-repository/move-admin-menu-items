== Description ==

Are annoyed of an unorganized and overcrowded admin menu? Do you wish to provide a well-arranged admin menu to the users of the WordPress backend? In this case, we have the perfect solution for you: Move all of your admin menu items, that you donâ€™t use regularly, into a condensed and clear admin menu.

== Installation ==

1. Upload the `stb_mami` folder into to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
